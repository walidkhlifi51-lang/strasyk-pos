# Strasyk POS

Application de caisse, livraison et commande en ligne, exécutable entièrement en local.

## Démarrage

1. Installer les dépendances : `npm install`
2. Lancer le serveur de développement : `npm run dev`
3. Construire l'application : `npm run build`

## Notes

- Les données locales sont stockées dans le navigateur.
- Les intégrations serveur peuvent être reconnectées plus tard sur votre propre infrastructure.
